Política de Privacidade — SoilCapture

Última atualização: 25/02/2026

O aplicativo SoilCapture é uma ferramenta acadêmica e científica destinada à captura padronizada de imagens de amostras de solo para fins de pesquisa.

Dados coletados

  • Imagens de amostras de solo, capturadas pela câmera do dispositivo.
  • Identificador da amostra (sample_id) informado pelo usuário.
  • Métricas técnicas da imagem, como data de captura e parâmetros básicos de qualidade.

O aplicativo não coleta dados pessoais identificáveis, como nome, e-mail, telefone, localização precisa ou dados biométricos.

Uso dos dados

  • Funcionamento da aplicação
  • Avaliação da qualidade das imagens capturadas
  • Fins acadêmicos e científicos

Armazenamento e compartilhamento

  • As imagens e os dados associados são enviados para um servidor controlado pelo desenvolvedor.
  • Os dados não são compartilhados com terceiros.

Retenção e exclusão dos dados

  • Os dados são armazenados apenas pelo tempo necessário para os objetivos acadêmicos.
  • O usuário pode solicitar a exclusão dos dados associados a uma amostra entrando em contato com o desenvolvedor.

Menores de idade

O aplicativo não é destinado a menores de 13 anos e não coleta intencionalmente dados pessoais de menores.

Segurança

São adotadas medidas técnicas razoáveis para proteger os dados contra acesso não autorizado.

Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Quaisquer alterações serão publicadas nesta página.
